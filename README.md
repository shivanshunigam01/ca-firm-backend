# CA Shivanand Choudhary Backend

Node.js + Express.js backend API for admin panel, user panel, leads, customers, quotations, content management, chatbot, and dashboard.

## Quick Start

```bash
cd server
npm install
cp .env.example .env
npm run seed
npm run dev
```

## Main API Base

`http://localhost:3001/api`

## Included Modules

- Admin authentication
- Leads management
- Customers management
- Quotations management
- PDF quotation download
- Email quotation sending
- Content management
- Dashboard stats
- AI chatbot

## Notes

- Configure MongoDB and environment variables before running.
- Configure OpenAI and SMTP credentials for chatbot and email features.
